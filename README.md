
# Airport finder

[Find your airport](https://airportfinder.herokuapp.com/)


### Development

Go to react-ui folder:

`cd react-ui`

*Development*

`npm run start`

*Build*

`npm run build`

*Test* and *coverage*
`npm run test`
`npm run coverage`

### Project

#### Client: 
1. React and Redux
2. Create react app tool was used to create the project.
3. Webpack for build

#### Server:
1. Express

#### Test:
1. Jest
2. Enzyme

[Heroku](https://devcenter.heroku.com/) to continous building and deploying
